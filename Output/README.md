# Output Folder

This folder contains sample outputs generated by the **Smart Silo Monitoring and Decision Support System**.

## Contents

The outputs include real-time monitoring reports generated by the ESP32-based smart silo system.

### Parameters Displayed

- Silo Level (cm)
- Temperature (°C)
- Humidity (%)
- Gas Concentration
- LDR Intensity
- IR Motion Status
- Demand Score
- Rain Probability
- Spoilage Status
- Market Advice
- Gate Status


## Description

The generated report provides a comprehensive overview of the storage conditions inside the grain silo and offers intelligent recommendations for stock management based on environmental and market-related factors.

These outputs can be used for:
- Monitoring grain storage conditions
- Detecting possible spoilage risks
- Assisting in stock selling decisions
- Ensuring storage security
